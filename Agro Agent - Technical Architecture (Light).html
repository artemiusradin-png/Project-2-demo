08 · Міжагентна взаємодіяС. 07 / 07

ПитанняРішення
Потік подійExpert генерує події діагнозу/рецепту → Operator споживає, генерує події транзакцій → Analytic агрегує дані Expert + Operator + SF. Одна append-only таблиця, один екземпляр PostgreSQL, без брокера повідомлень.
Ізоляція запису в SFЛише Operator пише (тільки чернетки). Expert, Analytic — лише читання. Три окремі набори дозволів.
Ізоляція відмовЖоден агент не в критичному шляху іншого. Operator не працює → Expert продовжує діагностику; ELT Analytic збоїть → Expert, Operator не зачеплені.
Межа зворотного зв'язку v1Зворотний зв'язок Expert (перенавчання / навчання релевантності) не автоматичний у v1. Замкнений цикл закривається через дію людини (огляд агрономом → оновлення каталогу/залишків). Автоматичний зворотний зв'язок відкладено на пост-v1.
ФазуванняExpert виходить у v1 (фундамент). Operator, Analytic підключаються після v1 як окремі етапи.
